# Ohrka::Feed

Provides an RSS feed for http://www.ohrka.de/ until they provide one

## Installation

Add this line to your application's Gemfile:

    gem 'ohrka-feed'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ohrka-feed

## Usage

See bin script for now.
